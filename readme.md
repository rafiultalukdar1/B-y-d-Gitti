## BüyüdüGitti – Kids Marketplace UI

BüyüdüGitti is a modern and fully responsive second-hand marketplace website designed for buying and selling pre-loved kids’ products. Built with HTML, CSS, and JavaScript, it delivers a clean and user-friendly interface inspired by real-world platforms, ensuring a smooth and engaging experience across all devices.

The project features a well-structured marketplace-style UI with product showcase sections, feature highlights, testimonials, and an interactive FAQ, all designed with a focus on usability, clarity, and consistent design patterns.

This project demonstrates my ability to build professional, user-focused websites using clean, maintainable code and strong responsive design principles.
